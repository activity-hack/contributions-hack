# GitHub Contributions Hack

## Allows you to write arbitrary messages in your GitHub contributions area

![screenshot](screenshot.png "Screenshot of program result")

## Usage Instructions

1. `git clone https://github.com/jeffslofish/contributions-hack.git`
1. `cd contributions-hack`
1. `python commit.py yyyy mm dd` 
(Replace yyyy mm dd with your own year, month, and day for the first date of your first letter. Should be a Sunday.)

## Notes
- Currently the only letters that are defined are those in the word "HELLO". Feelf free to add more and submit a pull request!
